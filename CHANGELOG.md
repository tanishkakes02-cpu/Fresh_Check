## [1.0.0] - 2024-03-24

### Added
- Initial release of FreshCheck
- Splash screen with fade animation
- Home screen with item list
- Add new item functionality with date picker and category selection
- Smart filtering: All, Expired, This Week, Safe
- Summary cards showing expired, expiring soon, and safe items
- Real-time status indicators with color coding
- Item deletion functionality
- Responsive dark theme UI with cyan accents
- Support for multiple item categories (Food, Medicine, Document, Cosmetics, Dairy, Other)

### Features
- Animated splash screen with 3-second delay
- Dynamic item status calculation (days left, color, text)
- Category-specific icons for items
- Beautiful dark UI with cyan accent color
- Smooth transitions and animations
- Bottom sheet modal for adding new items
- Scrollable item list with status and expiry information
